querySelectorAll

querySelectorAll is supported in modern browsers and allows you to select elements using CSS selectors (even complex ones).

querySelectorAll gives you a static node

getElementsByClassName

getElementsByClassName gives you a live node list

Typically faster in performance for selecting elements by class name compared to complex CSS selectors used in querySelectorAll

The terms "static node list" and "live node list" refer to how the collection of nodes retrieved by JavaScript methods behaves when the underlying DOM (Document Object Model) changes.
